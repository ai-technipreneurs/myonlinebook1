Objets ou conteneurs Iterable
#

Python propose des conteneurs.

Généralement, un conteneur, comme son nom l’indique, est un objet composite destiné à contenir d’autres objets. Ce notebook ne donne pas seulement des détails sur les listes et les dictionnaires, mais donne également un bref aperçu sur les ensembles et les tuples.

x='rooo7oolll'
list(x)

['r', 'o', 'o', 'o', '7', 'o', 'o', 'l', 'l', 'l']

Comme indiqué dans les sections précédentes, un programme python fonctionne toujours sur la base de la manipulation d’objets. Les instructions (qu’elles soient définies dans un code libre, dans une instruction if ou dans une boucle while ou for) sont toujours exécutées sur des objets. Les variables, comme nous l’avons vu précédemment, représentent un type particulier d’objets en python. Mais d’une manière générale, les instructions d’un programme python sont définies à partir d’une collection d’objets qui se présentent sous différentes formes de séquences de valeurs. Ces séquences de valeurs peuvent être constituées par exemple par un ensemble de variables à valeur unique. Cela signifie enfin qu’une variable définie à partir d’une valeur unique n’est généralement pas suffisante pour former l’architecture d’un programme. En python, nous avons :

  • Une liste d’objets est une séquence de valeurs (numériques et/ou caractères) indexées et spécifiées entre parenthèses, séparées par des virgules. Exemple :

       x = [ 1 , 5 , 8, 10 ,4 , 50, 8 ] # List that only contains numbers
   y = ["Olivier","ENGEL","Strasbourg"] # List that only contains characters
   z = [1, "Olivier",5 , 8, "ENGEL",10, 4, 50, "Strasbourg"] # List that contains numbers and characters

Un objet liste peut être généré manuellement en indiquant les valeurs entre parenthèses ou automatiquement par l’utilisation de la fonction list(). python         x = list(range(1,10))         print(x) # returns [1, 2, 3, 4, 5, 6, 7, 8, 9]      #

Un tuple d’objet est une séquence de valeurs (numériques et/ou caractères) indexée et spécifiées entre parenthèses, séparées par des virgules. Exemple : ```python x = ( 1 , 5 , 8, 10 ,4 , 50, 8 ) # tuple that contains numbers y = (“Olivier”,”ENGEL”,”Strasbourg”) # tuple that contains characters z = (1, “Olivier”,5 , 8, “ENGEL”,10, 4, 50, “Strasbourg”) # tuple that contains numbers and characters.

  ``` 
 Un objet **tuple** peut être généré manuellement en indiquant les valeurs entre parenthèses ou 
 automatiquement par l'utilisation de la fonction `tuple()`.
  ```python
    x = tuple(range(1,10))
    print(x) # returns (1, 2, 3, 4, 5, 6, 7, 8, 9)
 ```

  • Un objet ensemble est une séquence de valeurs (numériques et/ou caractères) non dupliquées, non indexées et spécifiées entre accolades, séparées par des virgules. Exemple :

       x = { 1 , 5 , 8, 10 ,4 , 50, 8 } # set that only contains numbers
   y = {"Olivier","ENGEL","Strasbourg"} # set that only contains characters
   z = {1, "Olivier",5 , 8, "ENGEL",10, 4, 50, "Strasbourg"} # set that contains both numbers and characters

    Un objet set peut être généré manuellement en indiquant les valeurs entre accolades ou automatiquement par l’utilisation de la fonction set().

       v = [2 , 4 , "orange", "viande", 4, "orange"]
   V = set(v)
   print(V) # returns {2, 'orange', 4, 'viande'}

  • Un objet dictionnaire est une séquence de valeurs (numériques et /ou caractères) indexées par des clés, spécifiées entre accolades et séparées par des virgules. A chaque clé correspond une ou plusieurs valeurs. Un dictionnaire est constitué d’un ensemble clés-valeurs. Exemple :

       x = {'name':'Jean', 'weight':70, 'height':1.75} # Keys with unique values
   y = {'Jean':[25,70,1.75],'Paul':[30,65,1.80],'Pierre':[35,75,1.65]} # Keys with multiple values formed by a list

Pour le dictionnaire x, les clés sont le nom, le poids et la taille. Les valeurs correspondantes de ces clés sont respectivement Jean, 70 et 1,75. Pour le dictionnaire y, les clés sont Jean, Paul et Pierre tandis que les valeurs sont des listes à trois valeurs (âge, poids et taille).

Un objet dictionnaire peut être généré manuellement en indiquant les valeurs entre accolades ou automatiquement par l’utilisation de la fonction dict() pour créer un dictionnaire vide. Nous reviendrons plus tard avec plus de détails sur la définition des dictionnaires.

Les quatre objets que nous venons de mentionner sont aussi ce que nous appelons des conteneurs pour la simple raison qu’ils peuvent contenir quelque chose. Ainsi, les chaînes de caractères, les listes, les tuples et les dictionnaires sont des objets itérables de base en Python. Les listes et les dictionnaires sont mutables - leurs composants peuvent être modifiés à la volée - alors que les chaînes de caractères et les tuples ne sont pas mutables.

Les trois derniers objets que nous citons ne sont pas itérables mais nous les mentionnons et les étudierons en détail dans la suite.

  • Un objet fonction est un programme conçu pour effectuer une opération précise. Par exemple la fonction print() permet d’afficher sur l’écran les résultats de l’instruction que nous lui fournissons ;

  • Un objet classe est une collection de fonctions, c’est-à-dire une association de fonctions connexes ;

  • Un objet module est une collection de classes, c’est-à-dire une collection de classes apparentées.

Operations sur les lists#

Une liste étant définie, on peut faire de nombreuses opérations afin de modifier sa structure ou la structure de ses composants.

  • Pour connaître le nombre d’éléments d’une liste, on utilise la fonction len(). Exemple : Soit x la liste définie comme suit :

   x = ['Monday', 'Tuesday', 'Wednesday', 1800, 20.357, 'Thursday', 'Friday']

Pour connaître la longueur de x (nombre d’éléments de x), on fait :

   print(len(x)) # this returns 7

  • On peut ajouter deux listes pour en former une autre en utilisant l’opérateur + qui permet la concaténation entre ces listes :

   x = ['giraffe','tiger']
   y = ['monkey','mouse']
   z = x + y
   print(z) # returns ['giraffe', 'tiger', 'monkey', 'mouse']

  • On peut multiplier les éléments d’une liste en utilisant l’opérateur de multiplication *.

   x = ['giraffe', 24, 18 , 'tiger', 2400, 150 ]
   y = x*3
   print(y) # returns ['giraffe', 24, 18, 'tiger', 2400, 150, 'giraffe', 24, 18, 'tiger', 2400, 150, 'giraffe', 24, 18, 'tiger', 2400, 150]

  • Il est possible de modifier un élément particulier d’une liste en utilisant son index.

   x = ['Monday', 'Tuesday', 'Wednesday', 1800, 20.357, 'Thursday', 'Friday']
   x[3] = x[3]+100
   print(x) # returns ['Monday', 'Tuesday', 'Wednesday', 1900, 20.357, 'Thursday', 'Friday']
   
   x[6] = 'Holy '+ x[6]  # pay attention on the space in 'Holy ' otherwise we will get HolyFriday
   print(x) # returns ['Monday', 'Tuesday', 'Wednesday', 1900, 20.357, 'Thursday', 'Holy Friday']

  • Il est possible d’ajouter de nouveaux éléments à une liste qui en contient déjà. Cela se fait à l’aide de la fonction append().

   x = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']
   x.append( ' Saturday ')
   x.append( ' Sunday ')
   print(x)

Comme on peut le voir, la fonction append() ne peut ajouter qu’un seul élément à la fois à une liste. C’est pourquoi nous pouvons utiliser la fonction extend() lorsque nous voulons ajouter à une liste plusieurs éléments en même temps. Exemple :

   x = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']
   x.extend(['Saturday','Sunday'])
   print(x)

k = {'name':'Jean','biometry':[25,70,1.75],'score':(12,17,15),'rank':25}

Operations on dictionaries#

  • Pour accéder aux éléments d’un dictionnaire, on utilise les clés. La méthode keys() retourne la liste des clés du dictionnaire et la méthode values() retourne les valeurs.

   X= {'name':'Jean', 'age':25, 'weight':70, 'height':1.75}
   print(X.keys())  # returns ['name', 'age', 'weight', 'height']
   print(X.values()) # returns ['Jean',25,70,1.75]   
   print(X['name'])  # returns 'Jean"   
   x = {' Jean ':[25,70,1.75],' Paul ':[30,65,1.80],' Pierre ':[35,75,1.65] }
   print(x['Jean']) #returns [25, 70, 1.75]
   print(x['Jean'][0]) #returns 25
   print(x['Jean'][0 :2]) #returns [25, 70]

  • Ajouter ou modifier les clés ou les valeurs : on peut modifier un dictionnaire soit en changeant les clés et les valeurs existantes, soit en en ajoutant de nouvelles. On peut également supprimer des valeurs ou des clés d’un dictionnaire.

  x= {} #  Creates an empty dictionary vide. We could also use x=dict()
  x['name'] = 'Jean' # adds the key-value name and Jean to the initial dictionary x 
  x['biometry']= [25,70,1.75] # adds the key-value biometry and [25,70,1.75] to the dictionary x 
  x['biometry']= [30,70,1.80] # modifies the values of the key biometry by making a new definition
  x['biomtry'][0]= 2 # modifies the element of index 0 of the list of values that corresponds to the key biometry (previously defined)

   Y = {' Jean ':[25,70,1.75],' Paul ':[30,65,1.80],' Pierre ':[35,75,1.65] }
   del Y['Jean'] # deletes the key Jean with its corresponding values
   del Y['Jean'][0] # deletes the element of index 0 in the sequence of value corresponding to the key Jean. For a key with one value, one uses del x['keyname'] where keyname is the name of the key that has an unique value.

  • Pour renommer une clé dans un dictionnaire, on utilise la fonction pop() telle que définie dans l’exemple suivant :

x = {' Jean ':[25,70,1.75],' Paul ':[30,65,1.80],' Pierre ':[35,75,1.65] }
x['John'] = x.pop('Jean') # One renames the key  Jean as John
print(x)
